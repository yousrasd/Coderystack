To store data in local storage we can make use of SharedPreferences. It offers a storage mechanism to store data in key value pair within an xml file.
We could storage simlpe key value information that are not sensitive.
If we need to use it to store sensitive info, we have to encode it.

If we want to store more more structured data, for example a user profile info, we can use databases through libs like Room or SQLite.
It allows us to store more ocmplex information within a table encapsulated in a database and offers possibility to link tables together
through the use of foreign keys. If you ve already done some backend developement, this should sound familiar.

We can also store temporar data within the device's cache 




You'll find mainly four folders:
1-files/ - internal storage files
2-cache/ - cache files
3-shared_prefs/ - SharedPreferences XML files
4-databases/ - SQLite databases
